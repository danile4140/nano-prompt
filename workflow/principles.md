# 1. 文档说明

将英文原著文章拆解为可生成插图/绘本分镜提示词的工作流，目标是让儿童基于图片完整复述故事。本工作流针对 **Nano Banana 2** 优化，包含 5 个节点：

- **5.1 角色生成**：提取角色并生成可复用角色设定卡
- **5.2 场景生成**：提取场景并生成可复用场景设定卡
- **5.3 分镜生成**（按章节循环）：基于角色/场景生成分镜
- **5.3.5 分镜校验**：对 5.3 输出做对齐校验并直接输出修正后的 storyboard_assets
- **5.4 分镜调整**：根据人工调整意见对分镜微调

**核心理念**：故事完整性靠拆镜+多格构图保证；画面稳定性靠固定描述复用保证；复述价值靠单图高密度叙事保证。每张分镜应承担"独立可复述的叙事节拍"。

参考：[Nano Banana 官方提示词指南](https://nanobanana.io/zh/prompt-guide)

# 2. Nano Banana 2 适配原则

## 2.1 总体原则

**适合**：主体清晰、动作明确、场景明确、重点前置、短句化、稳定复用固定描述。
**不适合**：单 prompt 叠加过多任务、多动作链/因果链堆叠、角色描述前后不一致、只写氛围不写空间结构。

## 2.2 提示词总结构（重要信息前置）

①构图与镜头 → ②动作与关系（含气泡/图示/画中画里的文字或图标）→ ③场景（地点/时间/天气/布局）→ ④光影与色彩 → ⑤主体与角色 → ⑥风格与媒介 → ⑦约束条件。

## 2.3 分镜设计原则

1. **逐句对应**：原文每句必须能在分镜中找到对应画面。
2. **高密度叙事**：在角色/场景稳定的前提下，单格尽量承载更多原文信息（多格构图、对白气泡、思维气泡、画中画、角标）。
3. **节拍而非动作**：拆镜最小单位是"叙事节拍"，单格内可同时含主动作 + 对白 + 心理 + 关键道具 + 数量/时间标记。
4. **场景跳变才拆镜**：仅场景切换、时间跳跃、视角转换时必须拆镜；其他情况优先在同一镜内增加图层。**例外：§4 必拆充要条件优先于本条，命中任一必拆条件时不论场景是否跳变，直接拆镜。**
5. **构图主动选择**：每条分镜必须从"构图配方表"主动选择主构图，而不是默认单格。
6. **角色/场景复用优先**：固定描述不随意改写。
7. **空间逻辑优先**：互动必须明确写出室内/室外、前后景、朝向、距离、关键道具位置。
8. **比例自然**：人物、家具、道具有自然比例参照。
9. **可读性兜底**：元素过多导致主体被挤压时必须降密度或改分格。

## 2.4 构图配方表

每条分镜从下列构图中**主动选择一种主构图**，可叠加二级补充与装饰特效。**所有构图选择直接用自然语言写进 prompt**，表中 ID（T1/T2/S1 等）只是内部代号，**不得**写进 prompt。

### 一级主构图（10 种）

> 注：表中 ID（T1/T2/S1 等）仅为内部代号，与 character_assets 的角色 ID（C1/C2…）和 scene_assets 的场景 ID（S1/S2…）不同含义，禁止写进 prompt。

| ID | 名称 | 触发条件 | 画面结构 |
| --- | --- | --- | --- |
| T1 | 序列条 | 同时满足以下三条才触发：①**同一主体**贯穿所有子步骤；②各子步骤属于**同类重复动作**（都是"找东西"/都是"被吩咐"/都是"变形结果"）；③每个子步骤**叙事权重均等且低**，不存在情绪反差或因果转折。三条有一条不满足即不用序列条。 | **横排 3 个**连续小格（16:9 横版禁用竖排；禁止横排四格） |
| T2 | 前后对比 | "变成了 / 从…变得… / 原本…现在" | **左右二分**（16:9 横版默认）；上下二分仅适合平铺对比物、无角色的场景对比 |
| S1 | 地图路径 | 位移性动词链（从 A 到 B 再到 C） | 俯视地图 + 虚线脚印路径 |
| S4 | 同时异地 | "与此同时 / meanwhile / 另一边" | 左右或对角分割 |
| C2 | 主画面+画中画 | 关键道具被首次提及或被使用 | 主画面 + 角落道具特写小框 |
| M1 | 思维图标气泡 | "想要 / 不想要 / 选择 / 列举多项" | 主画面 + 云气泡内多个图标 |
| M2 | 梦境/想象 | "梦到 / 想象 / 幻想 / wished" | 主画面 + 大型云状气泡内场景 |
| D1 | 对话双格 | 两角色对话**且**满足以下任一升级条件：①空间不可同框（距离远/隔门隔墙/不在同一视野）②情绪反差需要各自特写（请求→拒绝/得意→崩溃等强转折）③≥3 回合来回（单格气泡塞不下）；**不满足升级条件的同框对话默认用单格+双气泡** | 左右两格 + 对话气泡 |
| I1 | 物品清单 | 一次性列出 3 件以上物品 | 道具平铺 + 英文标签 |

**兜底顺序**（以上触发条件全不命中时）：**第一兜底：C2（主画面+画中画）**，多承载一层信息，优先于单格；**最后兜底：单格主画面**，仅当同时满足 ①只有 1 个简单动作 ②无对白 ③无关键道具/无心理活动 ④无前后状态变化 ⑤前后相邻镜无"天然成对"语义，五条全满足才允许使用。
